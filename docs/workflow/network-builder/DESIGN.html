VirVM's real route to China is standard 163; its 专线GIA is a paid upgrade. Dropping VirVM into the jump slot yields a non-CN2-GIA entry — the honest, teachable trigger for the China guardrail, since every purpose-built jump box (BWH, DMIT) is already GIA.

3One affordance, three inputs accessibility · non-negotiable

A palette card is a native <button> that is also draggable. Button semantics are the baseline for touch and keyboard; drag is a progressive enhancement. A drop target only accepts a compatible provider — an exit-only box dragged onto the jump slot gets a rust "reject" flash. In the fan-out, dropping/activating an exit appends a new branch (a persistent "＋ add residential exit" target sits at the foot of the fan); each branch has its own remove ✕ / Delete.

7Traffic-metered pricing prices PM-pending

When VirVM lands on the canvas as an exit (its 专线/CN2-GIA residential plan), that branch's card grows an inline selector — no dialog. Two controls fold into one effective $/mo that feeds metric ①:

ControlOptionsEffect
Traffic packagechips 100 · 300 · 500 · 1000 GBsets the package base (PM-pending numbers, stamped).
Billing cycle1 mo · 3 mo −5% · 6 mo −10% · 1 yr −20% · 2 yr −25%applies the discount, shown as an effective $/mo so it compares to flat boxes.

The estimate shows the fold-in explicitly, e.g. 300 GB $18 × 12 mo −20% = $172.80/yr → $14.40/mo effective, then summed with the shared jump and any other exits. The tier + cycle structure is final; only the dollar amounts wait on the PM.

11Accessibility & responsive verification

  • Widths checked: 1440 (three-column) · 390 (stack) · 320 (smallest) — no horizontal overflow; wide SVGs/tables sit in overflow-x:auto containers; fan-out branches wrap.
  • Contrast: text on rust/teal/indigo uses the darker -ink pairs (WCAG AA), per the site convention.
  • Keyboard: place / append-exit / remove / toggle / chip / cycle are focusable with a 2px focus ring; role-incompatible placement is blocked and announced; nothing is drag-only.
  • Motion: drag ghost + slot highlight collapse under prefers-reduced-motion.
  • Honesty: estimated values tagged; benchmarked vs reference badged; PM-pending numbers wear a ⚑ph stamp — no number reads as measured unless it is.
